The only requirement that may not be already installed for deakin students is 
# npm install express body-parser

Otherwise this repo can be copied locally and from cd ~/*/UsernameResponses run 
# node server.js 

You will know it is successful if you see 
# Server is listening at http://localhost:3000

And then you can run http://localhost:3000

GET mockups point to /getData?getData={$username} as an endpoint 
POST mockups point to /submitData as an endpoint
